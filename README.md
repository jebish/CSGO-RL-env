# Start

**Any OS** (recommended):

```bash
python start.py
```

Or:

| OS | Command |
|----|---------|
| macOS / Linux | `bash start.sh` |
| Windows | double-click `start.bat` or run `start.bat` |

Needs Python 3 installed and on PATH.
